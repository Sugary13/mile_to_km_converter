# 🧮 Mile to Kilometer Converter (Tkinter App)

Una pequeña aplicación gráfica hecha en Python con Tkinter que convierte millas a kilómetros. Ideal para practicar interfaces gráficas (GUI) básicas y manipulación de entradas de usuario.

---

## 🖥️ Vista general

> El usuario ingresa una cantidad en millas y pulsa un botón para ver el equivalente en kilómetros.


---

## 🚀 ¿Cómo ejecutarlo?

1. Asegúrate de tener Python instalado.
2. Ejecuta el script:

```bash
python main.py
```

⚙️ ¿Cómo funciona?
El usuario introduce una cantidad en millas.

Al presionar el botón "Convert", la aplicación:

Toma el valor

Lo multiplica por 1.609

Muestra el resultado redondeado a 2 decimales en la misma ventana

📦 Tecnologías utilizadas
Python 3

Tkinter – librería estándar para GUI en Python
